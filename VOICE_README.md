# 🎙️ Голосовые возможности CV-сайта

Ваш CV-сайт теперь имеет **бесплатную голосовую озвучку** через Web Speech API!

## ✨ Что это даёт

### 🆓 Voice Intro (Web Speech API) — **БЕСПЛАТНО**
**Автоматическая озвучка CV** при входе на сайт через встроенную в браузер технологию
- ✅ 100% бесплатно — без API ключей
- ✅ Без backend — работает в браузере
- ✅ Мгновенно — нет задержек
- ✅ Приватность — данные не покидают браузер

> 📖 **Документация:** [FREE_VOICE_GUIDE.md](FREE_VOICE_GUIDE.md)

## ⚡ Быстрый старт (бесплатная версия)

```bash
# Запустить проект
npm run dev
```

**Откроется:** http://localhost:5173

Вы увидите:
1. Автоматическая озвучка начнётся через 1.5 секунды
2. Справа внизу — кнопка управления озвучкой

**Никаких настроек не требуется!** Работает сразу из коробки.

## 📊 Что выбрать?

### ✅ Web Speech API (текущая версия)
**Используйте если:**
- Личный проект / портфолио
- Нужна бесплатность
- Не хотите возиться с API ключами
- Демо и прототипы

**Качество:** Хорошее, зависит от браузера

### ⚠️ Платные API (xAI, ElevenLabs, OpenAI)
**Используйте если:**
- Коммерческий продукт
- Нужны эмоции и живость
- Критично премиальное качество
- Кастомные голоса

**Качество:** Отличное, профессиональное
